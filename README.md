-mysql은 데이터베이스(삭제,추가,정렬)

-입력한 값을 데이터베이스로 넣는다.

3. 목록관리 : No, 부서명, 직급을 입력합니다. 입력한 내역은 [출퇴근기록]시트에서 자료를 불러올 수 있습니다.
